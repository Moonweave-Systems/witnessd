# v2.3.0 release notes draft

Publication is operator-only. This file is the draft text for the GitHub
release after W18 validation and push.

## Summary

v2.3.0 makes witnessd durable and installable:

- W15: parallel team execution with schedule receipts and Depone-derived
  overlap proof.
- W16: overlapping lane reconciliation through merge lanes and Depone merge
  receipts.
- W17: journaled replay-resume with append-only attempts and re-derived skip
  decisions.
- W18: one-command init/run/verify, quickstart CI, and in-session skill
  guidance that reports Depone verdicts instead of session self-reports.

## Quickstart

```bash
python3 -m witnessd init --home .witnessd --depone-root ../depone
run_json="$(python3 -m witnessd run "write two independent files" --repo . --home .witnessd)"
run_dir="$(python3 -c 'import json,sys; print(json.loads(sys.argv[1])["run_dir"])' "$run_json")"
python3 -m witnessd verify "$run_dir" --home .witnessd
```

CI also runs:

```bash
WITNESSD_DEPONE_ROOT=../depone scripts/quickstart_check.sh
```
